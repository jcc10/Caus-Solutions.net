What Is RegEx

Regular Expression or RegEx is a way of finding text in a string or document, it is one of the hardest things to understand but here is a list of RegEx we have made since we could not find them.
Also go here to see some nice tools for learning and understanding RegEx.

(?:(([0-9]|0[0-9]|1[0-9]|2[0-3]):[0-5][0-9])(\s|)(am|pm|AM|PM|)((\s|)- |))+

In our ticketing system we needed to find out if a time range is valid, so we made this RegEx to check it accepts any of the following formats:

  • 9:00 - 3:30 PM
